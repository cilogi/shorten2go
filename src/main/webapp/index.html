Mobile-friendly, scalable, URL shortener

Use this shortener if you need short URLs which you expect people to type into a mobile phone, as the URL contains only lower-case letters.

The demo runs on Google App Engine and will scale up to large volumes.

This demo doesn't have a short domain, so the output is long. To run it yourself you'll need a short domain.
